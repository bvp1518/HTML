Wht is the Anchor tag in HTML?

Anchor Tag is use with its href attribute.It is creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.

1.It is used to specify the target link to download when the user clicks.

2. It is used to specify the language of the linked document.

3.It is used to specify the linked media.

Example-1. We can use anchor tag for redirection to another webpage

1.Redirection only available when its a full link with HTTP or HTTPs,Otherwise display error like this

Example-6. If we set target as "_top" so, It opens the linked document in the full body of the window.

target = "_top"

Example-7. If we set target as "_self" so, It opens the linked document in the same frame..

target = "_self"

Example-8. If we set target as "_parent" so, It opens the linked document in the parent frameset.

target = "_parent"

Example-9. If we set target as "_blank" so, It opens the link in a new window.

target = "_blank"
