What is Heading tag in HTML?

An HTML heading tag is used to define the headings of a page. There are six levels of headings defined by HTML. These 6 heading elements are h1, h2, h3, h4, h5, and h6; with h1 being the highest level and h6 being the least.

  1. <h1> is used for the main heading. (Biggest in size)
  2. <h2> is used for subheadings, if there are further sections under the subheadings then the <h3> elements are used.
  3. <h6> for the small heading (smallest one).
  4. Browsers display the contents of headings in different sizes. The exact size at which each browser shows the heading can vary slightly. Users can also adjust the size of the text in their browser.
