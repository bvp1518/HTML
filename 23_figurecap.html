What is <figcaption>: The Figure Caption element?

The <figcaption> HTML element represents a caption or legend describing the rest of the contents of its parent <figure> element.

Example of < figure > as below

1. Caption in between Paraghraph tag < p >

figure 5.5

2. Caption in between div tag < div >

figure 5.5

3. Caption in between lable tag < label >

4. Caption in between Figcaption tag < Figcaption >
