Global attributes

Global attributes are attributes common to all HTML elements; they can be used on all elements, though they may have no effect on some elements.

  1. Accesskey - When used it specifies a shortcut for activating an element e.g. press Windows key + F6
  2. class - Class attributes bind the elements with their CSS
  3. contenteditable - This attributes guides whether the element is editable or not
  4. contextmenu - Right-click on the context area to find the custom menu defined for that section
  5. data - Using this attribute one can add information to be used in the application directly
  6. dir - the direction of the text in case of multi-use of languages
  7. draggable - This attribute specifies whether an element is draggable or not
  8. dropzone - These attributes can allow elements to copy move or link to other elements
  9. hidden - It instructs the browser whether that element is not yet or is no longer useful now
  10. id - Specifies the function and characteristics of every unique ID made
  11. lang - This attribute corresponds to the language of the element
  12. spellcheck - Spellcheck simply instructs the browser to whether check the spelling and grammar or not
  13. style - This tag is used to add inline CSS for elements that override global CSS as well as any other properties allocated to that element.
  14. tabindex - This tabindex attribute specifies the order of the element
  15. title - This attribute can add more meaningful and extra information to the element
  16. translate - This HTML attribute specifies whether the content of an element should be translated while browsing a different language or not
