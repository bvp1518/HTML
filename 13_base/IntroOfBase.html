What is base tag in HTML

  • The HTML base tag is used to specify a base URI, or URL, for relative links.
  • This URL will be the base URL for every link on the page and will be prefixed before each of them.
  • For example, if the URL specified by the base tag is “www.xyz.com” and then every other URL on the page will be prefixed by, “www.xyz.com/”.

    • Important Points

    1. The base tag must be defined between the head tags.
    2. There can be a maximum of only 1 base tag in a page.

Use of Base tag in HTML

We can put image or any file by hyper referance with using a Base tag in between head tag
