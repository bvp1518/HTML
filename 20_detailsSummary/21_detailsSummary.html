What is Details - Summary Elemants in HTML?

The < details> HTML element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state.

A summary or label must be provided using the < summary> element.

A disclosure widget is typically presented onscreen using a small triangle which rotates (or twists) to indicate open/closed status, with a label next to the triangle.

The contents of the < summary> element are used as the label for the disclosure widget.
