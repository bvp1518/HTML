What is blockquote?

  1. The HTML element < blockquote > designates the content within the tag as an extended quotation. Often, indentation is used to visually represent this.

  2. This element's attributes include the global attributes.

  3. Note: The <blockquote> tag in HTML4.1 defines the long quotation i.e. quotations that span multiple lines. But in HTML5, the <blockquote> tag specify the section that is quoted from other source.

  4. Attribute: It contains a single attribute cite which is used to specify the source of the quotation.
