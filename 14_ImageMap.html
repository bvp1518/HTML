What is image mapping tag in HTML?

An Image Map refers to the clickable image having a clickable area that can be used to navigate to the various link to other web pages
or the specific section of the same web page.
The < map > tag can be used to define the image map & the clickable area inside the image can be defined with the < area > tag.
