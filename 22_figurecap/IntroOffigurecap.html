What is <figcaption>: The Figure Caption element?

  • The <figcaption> HTML element represents a caption or legend describing the rest of the contents of its parent <figure> element.

  • The figure tag is a way to group together content.

  • It sometimes can go alongside the figcaption tag which is a way to describe the image, code snippet, diagram or an illustration.

  • It was introduced in HTML5 and is supported in all major browsers.

  • The benefit of the figure tag is that it adds more meaning (semantics) to the HTML document when compared to the image tag.

  • The figure container is just a way to wrap together your image with its caption.

  • If you think of a book you sometimes see an image and along the bottom a caption to explain something to do with the image.

  • Within a figure element, you can have multiple media elements like images or code snippets as long as they are related.

  • This option element can be placed before or after the content inside the element.

  • There can only be one figcaption within the figure element.
