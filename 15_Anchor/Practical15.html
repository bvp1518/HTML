Example-6. If we set target as "_top" so, It opens the linked document in the full body of the window.

target = "_top"

Example-7. If we set target as "_self" so, It opens the linked document in the same frame..

target = "_self"

Example-8. If we set target as "_parent" so, It opens the linked document in the parent frameset.

target = "_parent"

Example-9. If we set target as "_blank" so, It opens the link in a new window.

target = "_blank"
