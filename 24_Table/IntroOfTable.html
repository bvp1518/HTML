What is table tag in HTML?

  • If u want to arrage ur data or u want to disp ur data in proper alignment without CSS for that we can use table
  • Table is collection of row and column
  • In table we can create row by using <tr> tag
  • In table we can create column by using <td> tag table data
  • A table allows you to quickly and easily look up values that indicate some kind of connection between different types of data,

    • Common HTML Table tags

    1. < tr > - represents rows
    2. < td > - used to create data cells
    3. < th > - used to add table headings
    4. < caption > - used to insert captions
    5. < thead > - adds a separate header to the table
    6. < tbody > - shows the main body of the table
    7. < tfoot > - creates a separate footer for the table
