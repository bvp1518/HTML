1.Example of Canvas hollow rectangle image

<canvas id="Canvas" width="100" height="100" style="border:1px solid #000000;"> </canvas>

  • With the help of canvas element we can create only area to draw the graphics

  • We need some javascript for draw the graphics in this area

2.Let we use some script for fill green color in hollow rectangle

<canvas id="newCanvas" width="100" height="100" style="border:1px solid #000000;"> </canvas>
<script> var c = document.getElementById('newCanvas'); var ctx = c.getContext('2d'); ctx.fillStyle = 'green'; ctx.fillRect(0, 0, 100, 100); </script>

3.Let we use some script for draw a line in hollow rectangle

<canvas id="myCanvas" width="100" height="100" style="border:1px solid #000000;"> </canvas>
<script> var c = document.getElementById("myCanvas"); var ctx = c.getContext("2d"); ctx.moveTo(0, 0); ctx.lineTo(200, 200); ctx.stroke(); </script>

4.Let we use some script for draw a circle in hollow rectangle

<canvas id="myCanvas1" width="100" height="100" style="border:1px solid #000000;"> </canvas>
<script> var c = document.getElementById("myCanvas1"); var ctx = c.getContext("2d"); ctx.beginPath(); ctx.arc(50, 50, 40, 0, 2 * Math.PI); ctx.stroke(); </script>

4.Let we use some script for draw a linear gradient red color in hollow rectangle

<canvas id="myCanvas2" width="200" height="100" style="border:1px solid #000000;"> </canvas>
<script> var c = document.getElementById("myCanvas2"); var ctx = c.getContext("2d");
var grd = ctx.createLinearGradient(0, 0, 200, 0); grd.addColorStop(0, "red"); grd.addColorStop(1, "white");
ctx.fillStyle = grd; ctx.fillRect(0, 0, 200, 100); </script>
