What is Form in HTML?


  • An HTML form is a section of a document which contains controls such as text fields, password fields, checkboxes, radio buttons, submit button, menus etc.
  • An HTML form facilitates the user to enter data that is to be sent to the server for processing such as
  • For sending client data to the server there are two methods :
    • name, email address, password, phone number, etc. .
    1. GET
    2. POST

Why use HTML Form

  • HTML forms are required if you want to collect some data from of the site visitor.

List of All < form > Attributes

  1. Accept-charset Specifies the character encodings used for f orm submission
  2. Action Specifies where to send the form-data when a form is submitted
  3. Autocomplete Specifies whether a form should have autocomplete on or off
  4. Eenctype Specifies how the form-data should be encoded when submitting it to the server (only for method="post")
  5. Method Specifies the HTTP method to use when sending form-data
  6. Name Specifies the name of the form
  7. Novalidate Specifies that the form should not be validated when submitted
  8. Rel Specifies the relationship between a linked resource and the current document
  9. Target Specifies where to display the response that is received after submitting the form
