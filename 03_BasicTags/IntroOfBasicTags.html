List of Basic tag of HTML

  1. < head >
    The head tag contains all the elements describing the document.

  2. < title >
    The title tag specifies the HTML page title, which is shown in the browser’s title bar.

  3. < body >
    The body tag is where you insert your web page’s content.

  4. < p >
    A paragraph tag is used to define a paragraph on a web page.

  5. Heading Tag => <h1> to <h6>
    The HTML heading tag is used to define the heading of the HTML document. The < h1 > tag defines the most important tag, and <h6> defines the least.

  6. <a >
    for link

  7. <b >
    to make bold text

  8. <strong >
    for bold text with emphasys

  9. <body >
    main HTML part

  10. <br >
    for break

  11. <div >
    it is a division or part of an HTML document

  12. <i >
    to make an italic text

  13. <img >
    for images in document

  14. <ol >
    is an ordered list,

  15. <ul>
    for an unordered list

  16. <li >
    is a list item in bulleted (ordered list)

  17. <span >
    to style part of text
